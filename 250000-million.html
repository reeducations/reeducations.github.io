Answer:

The population by 2021 will be 535897.

Step-by-step explanation:

2014 Population: 250000 ; 10% is 25000

2015 Population: 250000+25000=275000 ; 10% is 27500

2016 Population: 275000+27500=302500 ; 10% is 30250

2017 Population: 302500+30250=332750 ; 10% is 33275

2018 Population: 332750+33275=366025 ; 10% is 36602.5

2018 Population: 366025+36602.5=402627.5 ; 10% is 40262.75

2019 Population: 402627.5+40262.75=442890.25 ; 10% is 44289.025

2020 Population: 442890.25+44289.025=487179.275 ; 10% is 48717.9275

2021 Population: 487179.275+48717.9275=535897.2025

Round off to whole number: 535897
